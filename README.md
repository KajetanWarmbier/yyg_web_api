# YYG WEB API
Beta version of backend and frontend for adding levels to YYG Android game.

## Frontend
**React.js**
* Axios
* Tailwindcss
* Headless UI
* React-Icons
* React-router-dom

## Backend
**Node.js**
* cors
* bcrypt
* dotenv
* express.js
* nodemon
* JsonWebToken
* mongoose
* morgan
* multer

## Design
* Concept design created in **Figma**
(**Proper design not yet implemented**)
